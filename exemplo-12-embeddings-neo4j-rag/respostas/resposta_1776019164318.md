Para converter objetos JavaScript em tensores no TensorFlow.js, você pode usar a função `tf.tensor()`. Tensores são a estrutura de dados fundamental em TensorFlow, e eles podem representar dados em várias dimensões, como escalares, vetores, matrizes e tensores de ordem superior.

Vamos ver um exemplo prático. Suponha que você tenha um array de números que deseja converter em um tensor. Aqui está como você pode fazer isso:

```javascript
// Importando o TensorFlow.js
import * as tf from '@tensorflow/tfjs';

// Criando um array de números
const array = [1, 2, 3, 4, 5];

// Convertendo o array em um tensor
const tensor = tf.tensor(array);

// Exibindo o tensor
tensor.print();
```

No exemplo acima, criamos um array simples de números e o convertimos em um tensor usando `tf.tensor()`. O método `print()` é utilizado para exibir o conteúdo do tensor no console.

Se você tiver um objeto mais complexo, como um array bidimensional (uma matriz), você pode fazer o seguinte:

```javascript
// Criando uma matriz 2D
const matriz = [
  [1, 2, 3],
  [4, 5, 6]
];

// Convertendo a matriz em um tensor
const tensor2D = tf.tensor(matriz);

// Exibindo o tensor 2D
tensor2D.print();
```

Aqui, a matriz é convertida em um tensor 2D. O TensorFlow.js automaticamente infere as dimensões do tensor com base na estrutura do array que você forneceu.
