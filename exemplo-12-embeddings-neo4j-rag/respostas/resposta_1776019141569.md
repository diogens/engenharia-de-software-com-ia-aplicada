Para converter objetos JavaScript em tensores usando TensorFlow.js, você pode utilizar a função `tf.tensor()`. Essa função permite que você crie tensores a partir de arrays ou arrays aninhados, que são estruturas comuns em JavaScript.

Por exemplo, se você tiver um array simples de números, pode convertê-lo em um tensor da seguinte forma:

```javascript
// Importando TensorFlow.js
import * as tf from '@tensorflow/tfjs';

// Criando um array de números
const array = [1, 2, 3, 4, 5];

// Convertendo o array em um tensor
const tensor = tf.tensor(array);

// Exibindo o tensor
tensor.print(); // Isso mostrará: [1, 2, 3, 4, 5]
```

Se você tiver um array bidimensional (ou seja, uma matriz), a conversão é igualmente simples. Veja o exemplo abaixo:

```javascript
// Criando um array bidimensional
const array2D = [[1, 2, 3], [4, 5, 6]];

// Convertendo o array bidimensional em um tensor
const tensor2D = tf.tensor(array2D);

// Exibindo o tensor
tensor2D.print(); // Isso mostrará: [[1, 2, 3], [4, 5, 6]]
```

Além disso, você pode especificar o tipo de dado do tensor usando o parâmetro `dtype`. Por exemplo, se você quiser criar um tensor de números inteiros, pode fazer assim:

```javascript
const intArray = [1, 2, 3, 4];
const intTensor = tf.tensor(intArray, undefined, 'int32');
intTensor.print(); // Isso mostrará: [1, 2, 3, 4]
```

Esses exemplos mostram como é fácil converter dados de JavaScript em tensores, que são a base para realizar operações de machine learning com TensorFlow.js. Se você tiver mais perguntas ou precisar de mais exemplos, fique à vontade para perguntar!
